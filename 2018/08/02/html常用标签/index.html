HTML的常用标签

关于HTML的标签 MDN 和 W3c 都有详细的文档说明，这篇博客只是对部分的常用标签做一个归纳。需要查询具体的参数，请移步到上面的官网；
另注:下文的标签属性均为html5所用属性，已废弃属性不在齐列。

<div>

如果说最常用的标签，<div>绝对是毫无异义的一位选手。

  • 关键词
    块级元素
  • 定义和用法
    在<div>标签定义的划分或HTML文档中的一个部分。
    在<div>元素通常被用作一个容器，用于其他HTML元素将它们与CSS样式或使用JavaScript执行某些任务。
  • 默认css设置
    div{
display:block;
}

<a>

  • 关键词
    超链接
  • 定义和用法
    <a>标签定义了一个超链接，其被用于连接从一个页面到另一个。
    <a>元素最重要的属性是href属性，这表示链接的目标。
  • 属性
属性 值 描述
download filename 规定被下载的超链接目标
href url 规定链接指向的页面的 URL
hreflang language_code 规定被链接文档我的语言
media media_query 规定被链接文档是为何种媒介/设备优化的
rel text 规定当前文档与被链接文档之间的关系
target _blank
_parent
_self
_top
framename		 规定在何处打开链接文档
type MIME type 规定被链接文档的MIME类型
  • 默认css设置
    div{
display:block;
}

<span>

  • 关键词
    行内元素
  • 定义和用法
    <span> 标签被用来组合文档中的行内元素.span 没有固定的格式表现。当对它应用样式时，它才会产生视觉上的变化。

<h1>~<h6>

  • 关键词
    标题
  • 定义和用法
    在<h1>到<h6>标记被用来定义HTML标题。
    <h1>定义的最重要的标题。<h6>定义最不重要的标题。

<p>

  • 关键词
    段落/块级元素
  • 定义和用法
    表示文本的一个段落。该元素通常表现为一整块与相邻文本分离的文本，或以垂直的空白隔离或以首行缩进。
  • 默认css设置
    p {
  display: block;
  margin-top: 1em;
  margin-bottom: 1em;
  margin-left: 0;
  margin-right: 0;
}

<ul>,<ol>,<li>

  • 关键词
    列表
  • 定义和用法
    1. <ul>代表多项的无序列表，即无数值排序项的集合，且它们在列表中的顺序是没有意义的。
    2. <ol>表示多个有序列表项，通常渲染为有带编号的列表。
    3. <li>用于表示列表里的条目。它必须被包含在一个父元素里：<ol>,<ul>,<menu>

<dl>,<dt>,<dd>

  • 关键词
    定义列表
  • 定义和用法
    1. <dl>是一个包含术语定义以及描述的列表，通常用于展示词汇表或者元数据 (键-值对列表)。
    2. <dt>用于在一个定义列表中声明一个术语
    3. <dd>用来在一个定义列表中声明一个术语的描述

<form>

  • 关键词
    块级元素/表单
  • 定义和用法
    1. <form>标签用于为用户输入创建 HTML 表单。
    2. 表单能够包含 <input>元素，比如文本字段、复选框、单选框、提交按钮等等。
    3. 表单还可以包含 <menus>、<textarea>、<fieldset>、<legend> 和 <label> 元素。
    4. 表单用于向服务器传输数据。
  • 属性
属性 值 描述
accept-charset charset-list 服务器可处理的表单数据字符集
action URL 提交表单时向何处发送表单
autocomplete on
off		 是否启用自动填充
enctype 发送表单之前如何编码
method get
post		 发送form-data的http方法
name form_name 表单的名称
novalidate novalidate 提交表单时不进行验证
target _blank
_parent
_self
_top
framename		 规定在何处打开 action URL

<input>

  • 关键词
    输入框/空元素
  • 定义和用法
    1. 用于搜集用户信息。
    2. 根据不同的 type 属性值，输入字段拥有很多种形式。输入字段可以是文本字段、复选框、掩码后的文本控件、单选按钮、按钮等等。

<img>

  • 关键词
    图像/空元素
  • 定义和用法
    1. 元素向网页中嵌入一幅图像
    2. 必有属性src,alt;

<main>

  • 关键词
    主要内容/唯一
  • 定义和用法
    1. 规定文档的主要内容
    2. 在一个文档中，不能出现一个以上的 <main> 元素。<main> 元素不能是以下元素的后代：<article>、<aside>、<footer>、<header> 或 <nav>。

<nav>

  • 关键词
    导航/块级元素
  • 定义和用法
    含有多个超链接的区域，这个区域包含转到其他页面，或者页面内部其他部分的链接列表.

<footer>

  • 关键词
    页脚/块级元素
  • 定义和用法
    1. <footer> 标签定义文档或节的页脚。
    2. <footer>元素应当含有其包含元素的信息。
    3. 页脚通常包含文档的作者、版权信息、使用条款链接、联系信息等等。
    4. 您可以在一个文档中使用多个 <footer>元素。
